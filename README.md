# Retail-Discount-Analysis

## Project Overview
This project analyzes **daily retail discount usage trends** using Python.
It helps understand how discounts change over time and their effect on sales.

## Objectives
- Analyze daily discount trends
- Clean and preprocess retail data
- Visualize insights using charts
- Learn Python data analysis workflow

---

## Dataset
The dataset is stored in CSV format and includes:
- Date
- Product Category
- Original Price
- Discount (%)
- Final Price
- Units Sold

---

## Tools & Technologies
- Python
- Jupyter Notebook
- Pandas, NumPy
- Matplotlib, Seaborn

---

## How to Run
1. Clone the repository  
2. Open Jupyter Notebook  
3. Run the notebook step by step
